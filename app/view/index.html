Reilig for Radargrams is an automatic detection tool for grave-like responses in greyscale radargrams. The model was trained on examples of medieval graves and works best for these or similar. As a single object detector, the result will show only one bounding box around one response detected in the image. There is potential for additional responses in the image which are not detected, as such similar responses in the image should be identified through human confirmation.
